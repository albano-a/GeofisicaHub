Recursos

Aqui você encontra diversos recursos para auxiliar nos estudos e pesquisas em Geofísica.

Sites

  • Sociedade Brasileira de Geofísica - SBGf - Maior sociedade de Geofísica da América Latina.
  • GeoSBG - Serviço Geológico do Brasil - Site que possui diversos dados, informações e produtos do Serviço Geológico Brasileiro.
  • Companhia de Pesquisa de Recursos Minerais (CPRM)
  • Society of Exploration Geophysicists (SEG) - Uma das maiores sociedades de Geofísica do mundo.
  • GeoScienceWorld - Site de Publicações Geofísicas
  • International Centre for Global Earth Models (ICGEM) - Centro internacional que fornece modelos globais da Terra, especialmente modelos gravitacionais e magnéticos, para apoiar pesquisas em geociências e disciplinas relacionadas.
  • Museu de Minerais, Minérios e Rochas Heinz Ebert - Museu que possui diversas informações sobre centenas de minerais.

Softwares

  • Geosoft - Análise e interpretação de dados geocientíficos.
  • Agisoft Metashape (Fotogrametria) - Permite a criação de modelos tridimensionais a partir de fotografias
  • QGIS (Open Source GIS) - Software muito utilizado para análise espacial e mapeamento.
  • OpendTect - Software especializado em interpretação e visualização de dados sísmicos, facilitando a análise geofísica tridimensional.
  • Seismic Unix (SU) - Ferramenta para processamento e análise de dados sísmicos, amplamente utilizada na indústria da geofísica.
  • GMT (Generic Mapping Tools) - Conjunto de ferramentas para criar mapas e gráficos geofísicos, útil para visualização e análise de dados espaciais.
  • SimPEG (Simulation and Parameter Estimation in Geophysics) - Software especializado em interpretação e visualização de dados sísmicos, facilitando a análise geofísica tridimensional.
  • Python - Linguagem de programação especializada em análise de dados.

Bancos de Dados Geofísicos

Referências à bancos de dados que fornecem conjuntos de dados geofísicos para pesquisa e prática.

  • GeoMaps | ANP - Mapa da Agência Nacional do Petróleo (ANP) que contém diversas informações sobre bacias terrestres e marítimas.
  • GEBCO - Site que possui diversos dados de batimetria de diversas áreas do mundo.
  • IRIS (Incorporated Research Institutions for Seismology) - Fornece dados sísmicos globais e recursos educacionais, sendo uma fonte abrangente para estudos relacionados a terremotos.
  • USGS (U.S. Geological Survey) Earthquake Hazards Program - Catálogo de terremotos mantidos pelo USGS, oferecendo informações detalhadas sobre eventos sísmicos em todo o mundo.
  • NOAA Nactional Centers for Environmental Information (NCEI) - Oferece uma ampla gama de conjuntos de dados geofísicos, incluindo informações sobre o ambiente marinho, clima e geociências.
  • EM-DAT (Emergency Events Database) - Base de dados que registra informações sobre desastres naturais e tecnológicos em todo o mundo, facilitando estudos de impacto e resposta.
  • SEG Wiki - Data Repositories - Lista de repositórios de dados geofísicos mantida pela SEG, abrangendo várias disciplinas da geofísica.
  • NGDS (National Geothermal Data System) - Exploração de dados geotérmicos, oferecendo acesso a conjuntos de dados relacionados à energia geotérmica.
  • NASA Earthdata - Portal da NASA para acesso a uma ampla variedade de dados terrestres, incluindo observações por satélite e informações ambientais.
  • Geoscience Australia - National Mineral and Fossil Collection Database - Banco de dados abrangente contendo informações sobre a coleção nacional de minerais e fósseis da Geoscience Australia.
